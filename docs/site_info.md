# Описание статического сайта

Данный сайт является основным результатом учебной практики и представляет собой информационный ресурс, посвященный проекту по дисциплине «Проектная деятельность».

## 1. Тематика проекта
Сайт разработан для презентации проекта **«Изучение русского языка как иностранного»**. 
*Основная цель ресурса* — предоставить пользователям информацию о целях проекта, достигнутых результатах и способах взаимодействия с командой разработчиков.

## 2. Структура и навигация
Сайт состоит из нескольких логических разделов:
*   **Главная страница (`index.html`)**: Краткое введение в суть и актуальность проекта и навигационное меню.
*   **О проекте**: Описание концепции проекта, его задач.
*   **Участники**: Информация об участниках проекта, с кем я непосредственно работала на протяжении семестра.
*   **Новости**: Раздел с постами, иллюстрирующими процесс выполнения проекта.
*   **Новости**: Раздел с материалами, позволяющими узнать больше о сути проекта.
*   **Контакты и Обратная связь**: Раздел с контактными данными и ссылкой на интерактивного Telegram-бота для сбора отзывов.

## 3. Технологический стек
При разработке сайта использовались стандартные веб-технологии, обеспечивающие высокую скорость загрузки и кроссбраузерность:
1.  **HTML5**: Семантическая разметка контента.
2.  **CSS3**: Стилизация интерфейса, использование Flexbox/Grid для расположения элементов.
3.  **Markdown**: Для ведения проектной документации и отчетов в репозитории.

## 4. Особенности реализации.
*   **Интеграция с вариативной частью**: На страницах сайта размещены активные ссылки и кнопки, ведущие в Telegram-бота обратной связи, что позволяет реализовать двустороннюю коммуникацию с пользователем без использования сложных серверных решений.

## 5. Инструкция по просмотру
Для ознакомления с сайтом достаточно открыть файл `index.html` в любом современном браузере.
