# Вариативная часть: Telegram-бот для обратной связи

В рамках выполнения вариативной части задания был разработан Telegram-бот, интегрированный со статическим сайтом для обеспечения интерактивного взаимодействия с пользователями.

## 1. Назначение бота
Бот выступает в роли модуля обратной связи. Он позволяет посетителям сайта:
* Оставлять отзывы о работе веб-ресурса.
* Задавать вопросы разработчикам.
* Получать мгновенное подтверждение о том, что их сообщение принято к рассмотрению.

## 2. Функциональные возможности
* **Прием сообщений**: Бот обрабатывает текстовые сообщения от пользователей.
* **Уведомления администратора**: Каждое новое сообщение пересылается администратору (разработчику) в личные сообщения.
* **Интерфейс**: Простое меню с кнопкой «Оставить отзыв».

## 3. Техническая реализация
* **Язык программирования**: Python.
* **Библиотека**: [Укажи свою, например: aiogram или telebot].
* **Интеграция**: Ссылка на бота размещена в разделе «Контакты» на основном статическом сайте.

## 4. Как это работает (алгоритм)
1. Пользователь заходит на статический сайт.
2. Переходит по ссылке `t.me/имя_твоего_бота`.
3. Нажимает кнопку `/start` и отправляет свой отзыв.
4. Разработчик мгновенно получает уведомление в Telegram с текстом отзыва.
